                              1. 프로젝트 개요
================================================================================

1.1 프로젝트 목적
--------------------------------------------------------------------------------
본 프로젝트는 대규모 언어모델(LLM)과 하이브리드 검색 기술을 결합하여
기업 내부 지식을 효율적으로 관리하고 검색할 수 있는 차세대 지식관리시스템(KMS)을
개발하는 것을 목표로 합니다.

주요 목표:
- 다국어(한국어, 일본어, 영어) 자연어 질의 지원
- 그래프 기반 + 벡터 기반 하이브리드 검색으로 높은 검색 정확도 달성
- GPU 가속을 통한 고성능 실시간 응답
- 기존 이슈관리시스템(IMS)으로부터 지식 수집 및 RAG 연동
- 기업용 보안 및 접근 제어 기능

1.2 개발된 프로토타입
--------------------------------------------------------------------------------
본 프로토타입은 다음과 같은 주요 기능을 포함합니다:

[핵심 기능]
(1) 하이브리드 RAG 질의응답 시스템
    - 자연어 질문에 대해 관련 문서를 검색하고 AI가 답변 생성
    - 벡터 유사도 검색 + 그래프 관계 탐색 + BM25 키워드 매칭 결합
    - 쿼리 의도 자동 분류 (VECTOR/GRAPH/HYBRID/CODE)

(2) IMS(Issue Management System) 지식수집
    - TmaxSoft IMS 시스템에서 기술지원 이슈 데이터를 자동 수집
    - 수집된 이슈를 벡터 임베딩으로 변환하여 RAG 시스템과 연동
    - 실시간 수집 진행률 표시 (Server-Sent Events)
    - 자연어 기반 시맨틱 검색으로 유사 이슈 탐색
    - 수집 결과 캐싱으로 중복 수집 방지 및 성능 최적화

(3) 마인드맵/지식그래프 시각화
    - 문서에서 자동 개념 추출 및 관계 분석
    - D3.js 기반 인터랙티브 시각화
    - 지식 노드 간 관계 탐색

(4) 대화형 AI 어시스턴트
    - 멀티턴 대화 지원
    - 대화 이력 저장 및 분기 기능
    - 스트리밍 응답으로 실시간 답변 표시

(5) 문서 관리
    - PDF, Word, Excel, 이미지 등 다양한 포맷 지원
    - 자동 텍스트 추출 및 벡터 임베딩
    - 세션 문서 우선순위 기능

(6) 엔터프라이즈 기능
    - 역할 기반 접근 제어 (RBAC)
    - 감사 로그 (Audit Log)
    - 다중 인증 (MFA) 지원
    - SSO/OAuth 통합 (Google, Microsoft)

4.3 IMS 지식수집 모듈 아키텍처 (Hexagonal Architecture)
--------------------------------------------------------------------------------

  본 모듈은 TmaxSoft IMS(Issue Management System)로부터 기술지원 이슈 데이터를
  자동으로 수집하여 벡터 임베딩으로 변환한 후 RAG 시스템과 연동합니다.

  [주요 기능]
  - IMS 시스템 자동 로그인 및 이슈 검색
  - 이슈 메타데이터 및 본문 추출 (제목, 카테고리, 제품, 버전, 모듈, 고객 등)

  - 자연어 질의로 유사 이슈 탐색 가능
  - RAG 컨텍스트로 활용하여 AI 기반 기술지원 답변 생성

5.1 기술적 특징
--------------------------------------------------------------------------------
(1) 하이브리드 검색 기술
    - 단순 키워드 검색의 한계를 극복
    - 의미적 유사도(Semantic) + 키워드 매칭(BM25) + 관계 탐색(Graph) 결합
    - 검색 정확도 대폭 향상

(2) GPU 가속 추론
    - NVIDIA A100 GPU 8장 활용
    - 실시간 응답 가능 (< 3초)
    - 대용량 모델(9B~12B 파라미터) 운영

(3) 다국어 지원
    - 한국어, 일본어, 영어 동시 지원
    - 다국어 임베딩 모델 활용
    - UI 국제화(i18n) 완비

(4) 엔터프라이즈 보안
    - 역할 기반 접근 제어 (ADMIN/LEADER/SENIOR/USER/GUEST)
    - 감사 로그 자동 기록
    - 암호화된 자격증명 저장

(5) 확장 가능한 아키텍처
    - Hexagonal Architecture 적용
    - 마이크로서비스 분리 용이
    - 컨테이너 기반 배포

5.2 비즈니스 가치
--------------------------------------------------------------------------------
(1) 지식 검색 효율화
    - 기존 키워드 검색 대비 관련 문서 발견율 향상
    - 자연어 질문으로 원하는 정보 즉시 획득

(2) 기술지원 업무 효율화
    - IMS 이슈 지식을 자동 수집하여 RAG와 연동
    - 수집된 기술지원 이력을 바탕으로 AI가 유사 사례 기반 답변 생성
    - 유사 이슈 자동 추천으로 중복 해결 방지
    - 신규 이슈 대응 시간 단축

(3) 지식 시각화
    - 마인드맵으로 지식 구조 파악
    - 관계 그래프로 연관 정보 탐색

(4) 협업 강화
    - 대화 이력 공유 가능
    - 팀 단위 지식 베이스 구축
